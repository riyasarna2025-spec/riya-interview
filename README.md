Mandatory Features:

1. Homepage:

o Hero section with a full-screen background image or video of the athlete.

o Short bio with dynamic counters (eg., total wins, championships, social media followers).

o Navigation bar with smooth scroll to all sections.

2. Career Timeline:

Interactive timeline of milestones and achievements.

o Users can click on events to expand details or media (photos, videos, news links).
3. Stats & Achievements Section:

o Visual representation of stats (charts, graphs, badges).

o Include at least 3 interactive visual elements (like hover animations, filters, or sortable tables).

4. Media Gallery:

o Responsive gallery of images, videos, and interviews

o Filtering option (e.g, by year, event, or type of media).

o Clicking a media item should open a modal/lightbox

5. Social & Fan Engagement:

o Links to official social media accounts.

o Embed a Twitter/instagram feed or news updates dynamically (optional API usage).

o Optional fan form, users can leave a message or submit fan art

6. Responsive & Interactive Ul:

o Fully responsive on desktop, tablet, and mobile.

o Smooth animations and transitions for scrolling, hover effects, and media modals.

o Sticky navigation bar that highlights the current section.
